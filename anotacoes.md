# JavaScript

## Funcionalidades

- Criar algoritmos e programas para executar no navegador: client side
- Manipular o DOM: elementos HTML, eventos (clique, submit), estilos CSS
- Node.js: framework JS para back-end / runtime em JS
- Mongo.db / GraphQL: banco de dados em JS
- React / Vue.js / Angular: frameworks JS para desenvolvimento web / mobile
- React Native: framework JS para desenvolvimento mobile
- Receber e manipular dados
- Tomar decisões baseados na lógica computacional
- loop e interações
- condições de saída

## Execução

- Node.js Server;
- Navegador (todos possuem JS implementado)

## Requisitos

- Navegador com suporte a JS;
- Node.js, ambiente em tempo de execução multiplataforma para JavaScript.

## Variáveis

- var  
  var a=10
- const  
  const b=15
- let  
  *deve ser utilizado no lugar de var*
- Tipagem dinâmica
  - string
  - number
- array  
  let d = [10, 'texto', 10.5]  
  d[0] -> retorna o elemento 0 do array 'd'

## Operadores

- Soma +
- Subtração -
- Multiplicação *
- Dividir /
- Módulo %
- Math: random(), round(), sqrt().

## Atribuidores

x = y -> atribui o valor de 'y' à variável 'x'.

x += y -> soma 'x' com 'y' e atribui o resultado a 'x'.

x -= y

x *= y

x /= y

x %= y
